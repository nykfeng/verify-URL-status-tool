URL Status Verfication Tool

This is a business tool used to verify the current status of a list of brand URLs. The app will run through each URL, and return whether the URL is still working. It will return a status code and if the URL redirects, it will return the new URL.

Upload your Excel file with the following format:

Brand ID Brand Name Brand URL
123456 Apple, Inc. https://www.apple.com
789456 MediaRadar, Inc. https://www.mediardar.com
1753951 Netflix https://www.netflix.com
